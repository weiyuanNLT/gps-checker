GPS Checker & Claim Generator

Version: A1.1

The tool is arranged into 5 sections: Section 1 records manhole entry, Section 2 checks pipe length from GPS coordinates, Section 3 captures the pipe acceptance report checklist, Section 4 captures additional claims, and Section 5 generates the surface summary and SWP priced items.

Input all fields in the Project Information section, upload the GPS Coordinate File, then click Generate Table.

Project Information

Section 1: Dynamic Manhole Entry

Input the manhole connection details and select the related IPID No. for each entry.

Tick Boring Same Wall only for subsequent boring on the same wall face, not the first manhole. If a manhole number is marked as New MH, the same manhole number will be treated as New MH in later rows.

Section 2: Pipe Length

Review the calculated pipe length, then select the surface type and enter the common trench reference where applicable.

Section 3: Pipe Acceptance Report

Complete the pipe acceptance checklist and photo status information for the pipe testing record.

Section 4: Additional Claims

Tick any applicable claim and enter the quantity where required; blank or invalid quantities will default to 1 when selected.

Select Surface/Common Trench, complete Section 1, Section 3 and Section 4, then generate the summary and SWP priced items.

Section 5: Summary

Click Generate Summary and Quantities to produce the surface and laying summary together with the SWP priced items.
